# Fifa-player-analysis
Data analysis and machine learning project exploring FIFA player performance, wages, and potential.

## 📊 Project Overview
This project analyzes FIFA player data to understand how player performance, wages, and potential are related. The goal is to explore how players are evaluated and to use statistical methods and machine learning models to predict outcomes such as player potential and age category.


## ❓ Problem Being Solved
Soccer teams must decide how much to pay players and which players will improve over time. This project uses data to answer questions such as:
- What factors influence player wages?
- How much do players typically improve?
- Can player potential be predicted using current data?


## 📁 Dataset
The dataset used is **fifa_players.csv**, which includes:
- Age
- Overall rating
- Wage (EUR)
- Potential
- Other player attributes
